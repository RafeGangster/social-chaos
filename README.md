# 🔥 SOCIAL CHAOS — Random Social Challenge Generator

> *Real challenges. Real people. No excuses.*

A blazing-fast, single-file web app that generates hyper-specific social dares to push you out of your comfort zone and into real conversations with real people.

---

## 🎯 What Is This?

**Social Chaos** is a random challenge generator built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, no installs. Just open it and spin.

Every challenge is specific. Not *"talk to someone"* — but *"walk up to a girl in red and ask her about her family and where she grew up."* Real targets. Real topics. Real coaching tips on how to actually pull it off.

---

## ✨ Features

- 🎲 **120 unique challenges** — never runs dry
- 💀 **3 difficulty tiers** — Easy, Medium, and Hard
- 🔥 **Heat meter** — visual indicator of how spicy each dare is
- 💡 **Coaching tip** on every card — so you know exactly what to say and how
- 📊 **Live deck counter** — shows how many challenges are in the current filter
- 🕹️ **No-repeat logic** — won't give you the same dare twice in a row
- 📜 **History log** — tracks your last 5 spins
- ⚡ **Zero dependencies** — one HTML file, runs anywhere

## 🎮 How To Use

1. Open `social-chaos.html` in any browser
2. Pick a difficulty — **Easy**, **Medium**, or **Hard** (or leave it on All)
3. Hit **SPIN**
4. Read the dare and the coaching tip
5. Go do it. No skipping.
